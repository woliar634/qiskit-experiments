# Qiskit Experiments: Simulation to Dynamic Circuits

This repository documents my hands-on exploration of Qiskit, covering
basic simulation, primitives, and dynamic circuits executed on
IBM Quantum hardware.

## Contents
- Bell state ("Hello World") simulation
- Qiskit primitives (Sampler, Estimator)
- Dynamic CNOT circuits with classical feedforward
- Hardware results on IBM Torino
- Noise scaling with system size

## Key Learning
- Identified and fixed a classical-register measurement bug
- Observed Bell population decay with qubit count
- Understood limitations of population vs fidelity

## Tools
- Qiskit
- IBM Quantum Runtime
- Python, Matplotlib

## Status
Learning-focused experimental repository.
