## 📖 Overview

This repository documents my exploration of **quantum computing** using the **Qiskit SDK** and **IBM Quantum Platform**. It contains a collection of Jupyter notebooks demonstrating quantum circuit design, quantum primitives, dynamic circuits, state preparation, and execution on simulators and real quantum processors.

The objective of this repository is to build a practical understanding of quantum programming while experimenting with modern quantum hardware and software tools.

# 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/woliar634/qiskit-experiments.git
```

### Install the dependencies

```bash
pip install -r requirements.txt
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open any notebook and execute the cells sequentially.
